Module astronvim.utils

AstroNvim Utilities

Various utility functions to use within AstroNvim and user configurations.

This module can be loaded with local utils = require "astronvim.utils"

load_plugin_with_func(plugin, module, func_names)
A helper function to wrap a module function to require a plugin before running

Parameters:

  • plugin the plugin string to call require("lazy").laod with
  • module the system module where the functions live (e.g. vim.ui)
  • func_names a string or a list like table of strings for functions to wrap in the given moduel (e.g. { "ui", "select })

set_mappings(map_table, base)
Table based API for setting keybindings

Parameters:

  • map_table A nested table where the first key is the vim mode, the second key is the key to map, and the value is the function to set the mapping to
  • base A base set of options to set on every keybinding

Run a shell command and capture the output and if the command succeeded or failed
